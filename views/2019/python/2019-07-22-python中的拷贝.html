# python中的拷贝

python中的拷贝有，有引用，浅复制，深复制的区别（引用，copy，deepcopy）

In [40]: import copy

In [41]: List = [1, 2, 3, 4, ["a", "b"]]

In [42]: one_list = List

In [43]: seconde_list = copy.copy(List)

In [44]: third_list = copy.deepcopy(List)

In [45]: List.append(5)

In [46]: List[4].append("c")

In [47]: List
Out[47]: [1, 2, 3, 4, ['a', 'b', 'c'], 5]

In [48]: one_list
Out[48]: [1, 2, 3, 4, ['a', 'b', 'c'], 5]

In [49]: seconde_list
Out[49]: [1, 2, 3, 4, ['a', 'b', 'c']]

In [50]: third_list
Out[50]: [1, 2, 3, 4, ['a', 'b']]

In [51]: id(List), id(one_list), id(seconde_list), id(third_list)

浅拷贝，身拷贝，都是针对可变对象来说的，
引用是直接在原对象上创建一个引用
浅拷贝是新建一个对象，其中的元素使用的还是原对象的引用。
深拷贝则是新建一个对象，其中的元素使用的也是新建的对象。

浅拷贝是对引用的拷贝，深拷贝是对资源的拷贝

from copy import deepcopy

a = [1, 2]
b = a
c = deepcopy(a)

a.append(3)
print("a:", a, "b:", b, "c:", c)

str1 = "one"
str2 = str1
str1 = "onetwo"
print("str1:", str1, "str2:", str2)

a: [1, 2, 3] b: [1, 2, 3] c: [1, 2]
str1: onetwo str2: one

  1. 变量赋值是将一个对象的地址付给另一个变量。
  2. 修改不可变对象str，tuple需要开辟新空间
  3. 修改可变对象不需要开辟新空间
  1. 赋值：只是对原对象的引用，两个对象id相同。
  2. 浅拷贝：创建一个新的组合对象，和原组合对象共享子对象。
  3. 深拷贝：创建一个新的组合对象，并且递归的拷贝原组合对象的子对象。两个对象之间没有任何关系。实际上两者共享不可变子对象，但是修改自己的不可变对象不会影响到其他的对象。
    浅拷贝和深拷贝的区别仅仅是对组合对象来说的，组合对象就是包含了其他对象的对象，如列表，类实例。其他数字，字符串以及其他原子类型没有拷贝一说，产生的都是原对象的引用。
from copy import copy, deepcopy

a = [1, 'str', [1, 2]]

b = a
c = copy(a)
d = deepcopy(a)

if __name__ == '__main__':
    print(id(a), id(a[0]), id(a[1]), id(a[2]))
    print(id(b), id(b[0]), id(b[1]), id(b[2]))
    print(id(c), id(c[0]), id(c[1]), id(c[2]))
    print(id(d), id(d[0]), id(d[1]), id(d[2]))
