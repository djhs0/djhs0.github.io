          # 作用域的产生

          只有变量在module、Class、def中定义时才会产生作用域。
          for、while、try-except/finally、with等关键字语块中不会产生作用域。

          def varLocal():
    var_def = "xxx"

print(var_def)

# 无错误


if False:
    var_if_false = "false"

print(var_if_false)
# 有错误
# NameError: name 'var_if_false' is not defined


with open("test.txt", 'w') as f:

          全局作用域(gloable)
          全局作用域指的是在模块中定义的作用域。在模块顶层声明的变量是全局变量。模块中的全局变量仅限于模块中。

          # 函数闭包

          闭包函数的定义：函数返回了一个其内部定义的函数，那么被返回的函数就是闭包函数。
          函数的闭包：指得是闭包函数在返回时保留了其父函数的namespace，以至于其在父函数结束时，依旧能调用父函数定义的变量。
          闭包：之所以将closure翻译成闭包，猜测是虽然父函数关闭了整个关闭了，但是既自成一个命名空间，但是它又包含了关闭了的父函数的命名空间。

          outer_var = "50"
def outer(x):
    def inner():
        y = "2"
        print("inner gloable:{0}".format(outer_var))
        print("inner:{0}".format(x))
    return inner

inone = outer(1)
intwo = outer(2)
print(inone.__closure__)
print(intwo.__closure__)
inone()
intwo()
