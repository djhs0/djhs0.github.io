          重构2Go例子：以对象取代基本类型

          djhs0 2019-9-15 重构 go

          # 对象取代基本类型

          为了处理数据方便，各种现代高级语言都提供了简单好用的基本数据类型，比如字符串，整数，浮点数，布尔类型。但是在一些场景中直接使用这些类型并不是一种好的实践，在学习编程的初期，很容易被基本类型惯坏，在程序中可以使用基本类型的地方就懒得创建功能更强大，更易扩展的自定义对象。
          所谓以对象取代基本类型，就是将基本类型封装一下，虽然本质上还是基本类型提供值，但是自己定义的对象明显更易修改和扩展，实现更强大的功能。

          # 动机

          开发初期，由于业务前途方向不明朗，一些简单数据可以使用基本类型描述的话，工程师们也就懒得创建新的对象。随着项目越来越大，发现以前依托于基本类型描述的数据难以修改和扩展。
          所有人的手机号码可以看作一个字符串，当使用string来描述手机号码，可以很方便的使用数据，但是当需要其它方式打印时就需要编写一个方法来实现

          # 例子

          // MobileNumber 手机号码
type MobileNumber struct {
	value string
}

// NewMobileNumber 创建一个新手机号对象
func NewMobileNumber(number string) MobileNumber {
	m := MobileNumber{}
	m.SetNumber(number)
	return m
}

// SetNumber 设置手机号码
func (t *MobileNumber) SetNumber(value string) {
	t.value = value
}

// Value origin value
func (t *MobileNumber) Value() string {
	return t.value
}

// Value344 344格式的手机号
func (t *MobileNumber) Value344() string {
	var byte344 []byte
	if len(t.value) != 11 {
		panic("invalid mobile phone number")
	}
	for i, v := range t.value {
		if i == 3 || i == 7 {
			byte344 = append(byte344, '-')
		}
		byte344 = append(byte344, byte(v))
	}

	return string(byte344)
}

func main() {
	bpn := NewMobileNumber("17778902345")
	fmt.Println(bpn.Value())
	fmt.Println(bpn.Value344())
}
