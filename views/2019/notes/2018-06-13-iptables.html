    iptables简要概念

    djhs0 2018-6-18 iptables

    # 防火墙

    # 硬件防火墙和软件防火墙

    在防火墙分类中有硬件防火墙和软件防火墙，分别在物理上和软件上阻隔数据。

    # iptables

    iptables属于软件防火墙，iptables实际上并不具有防火墙的功能，其只是linux中防火墙框架netfilter提供的客户端命令行工具，iptables将用户制定的规则写入到netfilter中。用户能同过iptables对数据包过滤和修改。

    1. 当数据包通过进入系统内核之前首先要经过iptables的5条链。
    2. 5条链上制定着管理数据包的规则。
    3. 当数据包到达一个链时，iptables会逐一校验数据包是否符合规则。
    4. 若数据包不符合链中的任意规则，则iptables使用该链的默认规则处理它。

    # 四个表五条链

    iptables由4个表和5条链表示。

    # table类型

    表名 功能 优先级 内建链
    filter 数据包过滤 4
    nat 网络地址转换 3
    mangle 数据包修改，拆解修改重新组装 2
    raw 数据追踪 1

    # 链类型

    链类型 作用域
    PREROUTING 在进入路由表之前
    INPUT 数据包经过路由表之后目标为本机
    OUTPUT 本机向外
    FORWARD 通过路由表后，目的不为本机
    POSTROUTING 数据包发送到网卡接口之前
